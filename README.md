1. Add your MP4 file to the repository as `rickroll.mp4`.
2. Push to GitHub.
3. Enable GitHub Pages in repository settings using the `main` branch and the `/ (root)` folder.
4. The site will be available at `https://booooo69.github.io/basic-idea-behind-this-account../` or the URL shown in GitHub Pages settings.
